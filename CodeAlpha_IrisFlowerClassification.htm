    Iris Flower Classification

    Project Overview

    This project uses Machine Learning to classify Iris flowers into three species:

    • Iris Setosa
    • Iris Versicolor
    • Iris Virginica

    Dataset Features

    • Sepal Length
    • Sepal Width
    • Petal Length
    • Petal Width

    Technologies Used

    • Python
    • Pandas
    • Scikit-learn

    Algorithm Used

    Decision Tree Classifier

    Project Workflow

    1. Load the Iris dataset.
    2. Split the dataset into training and testing sets.
    3. Train the Decision Tree model.
    4. Predict flower species.
    5. Evaluate model accuracy.

    Output

    The model predicts the species of an Iris flower based on the given flower measurements.
